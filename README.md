💰 Smart Expense & Investment Planner for College Students
📌 Overview

The Smart Expense & Investment Planner is a Python-based command-line application designed to help college students manage their finances effectively. It enables users to track daily expenses, manage budgets, monitor savings, and receive basic investment suggestions.

This project aims to build financial awareness and encourage disciplined money management habits at an early stage.

🚀 Features
1. Expense Tracker
Add daily expenses
Categorize spending (Food, Travel, Entertainment, etc.)
View monthly total spending
2. Budget Management
Set a monthly budget
Track total spending
View remaining balance
3. Savings Tracker
Automatically calculates savings
Displays monthly savings
Tracks savings trends
4. Investment Suggestions
Recommends simple allocation strategy:
50% → Expenses
30% → Savings
20% → Investment
Suggests beginner-friendly options:
Mutual Funds
Stocks
Emergency Fund

⚙️ How It Works
📂 Data Storage
Uses a JSON file to store expense data
Ensures persistent storage
➕ Add Expenses
Input amount, category, and date
Stored as structured data
📊 View Expenses
Displays all recorded expenses
Shows date, amount, and category
📈 Financial Summary
Takes monthly income
Calculates:
Total expenses
Savings = Income − Expenses
Suggested investment (50% of savings)
Warns if expenses exceed income
📊 Category Analysis
Groups expenses by category
Helps identify spending patterns
🖥️ User Interface

Menu-driven system:

Add Expense
View Expenses
Financial Summary
Exit

Make sure Python is installed on your system.
