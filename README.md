# 💰 Smart Expense & Investment Planner for College Students

## 📌 Overview
The Smart Expense & Investment Planner is a Python-based command-line application designed to help college students manage their personal finances effectively.

It allows users to:
- Track daily expenses  
- Manage monthly budgets  
- Monitor savings  
- Get basic investment suggestions  

This project aims to promote financial discipline and help students build smart money habits early in life.

---

## 🚨 Problem Statement
Most college students:
- Spend money without tracking  
- Save very little or nothing  
- Lack knowledge of investments (stocks, mutual funds)  

### ❗ Result:
- Poor financial management  
- No savings habit  
- No long-term financial planning  

---

## 🎯 Objective
To build a system that:
- Tracks expenses  
- Encourages savings  
- Suggests simple investment strategies  
- Promotes financial awareness  

---

## ✨ Features

### 📊 Expense Tracker
- Add daily expenses  
- Categorize expenses (Food, Travel, Entertainment, etc.)  
- View all expenses  

### 💸 Budget Management
- Set monthly budget  
- Track total spending  
- Display remaining balance  

### 💰 Savings Tracker
- Savings = Income − Expenses  
- Shows monthly savings  
- Tracks savings trends  

### 📈 Investment Suggestions
- 50% → Expenses  
- 30% → Savings  
- 20% → Investment  

- Recommends:
  - Mutual Funds  
  - Stocks  
  - Emergency Fund  

---
